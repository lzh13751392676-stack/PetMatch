# 📱 手机端测试完整指南

## 方法一：Chrome 开发者工具（推荐）⭐⭐⭐⭐⭐

### 步骤：
1. 打开网页（运行 `启动服务器.bat`）
2. 按 `F12` 打开开发者工具
3. 按 `Ctrl + Shift + M` 切换手机模式
4. 顶部选择设备型号

### 常用设备：
- iPhone 12/13/14 Pro
- iPhone SE
- iPad Air
- Samsung Galaxy S20
- Pixel 5

### 功能：
- ✅ 测试不同屏幕尺寸
- ✅ 模拟触摸操作
- ✅ 测试横屏/竖屏
- ✅ 调整网络速度
- ✅ 查看控制台错误

---

## 方法二：真机测试（最准确）⭐⭐⭐⭐⭐

### 步骤：

#### 1. 运行 `手机测试.bat`
   - 自动显示本机IP地址
   - 自动启动服务器

#### 2. 手机连接同一WiFi
   - 确保手机和电脑在同一网络

#### 3. 手机浏览器输入地址
   - 例如：`http://192.168.1.100:8000`

### 手动获取IP地址：
```bash
# Windows
ipconfig

# 找到 "IPv4 地址" 那一行
# 例如：192.168.1.100
```

## 💡 常见问题

### Q: 手机无法访问？
A: 检查：
1. 手机和电脑是否同一WiFi
2. 防火墙是否阻止
3. IP地址是否正确

### Q: 触摸滑动不流畅？
A: 已添加 `{passive: false}` 优化

### Q: 如何调试手机浏览器？
A: 
- Android: Chrome远程调试
- iPhone: Safari开发者模式

---

## 🚀 快速开始

1. **电脑测试**：按 `F12` → `Ctrl+Shift+M`
2. **手机测试**：运行 `手机测试.bat`
3. **在线测试**：部署到 Netlify/Vercel

推荐先用 Chrome 开发者工具快速测试，再用真机验证！
